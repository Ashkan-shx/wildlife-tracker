Project Description

The Wildlife Sighting Tracker is a specialized Express.js web application designed to manage and display wildlife observation data[cite: 22]. It provides a robust API that allows users to filter sightings by verification status, habitat, and species, making it a powerful tool for environmental monitoring.

Available API Endpoints

Click the buttons below to fetch data directly onto this page, or open the links in a new tab to see the raw JSON response.

/api/sightings - Returns all wildlife sighting records.
/api/sightings/verified - Returns only sightings that have been verified.
/api/sightings/species-list - Returns a unique list of all species names.
/api/sightings/habitat/forest - Returns sightings specifically from the forest habitat.
/api/sightings/recent - Returns the 3 most recent wildlife sightings.
